# FAF-Tech

FAF-Tech adalah platform website teknologi yang menampilkan berbagai inovasi dan proyek yang melibatkan AI (Artificial Intelligence), pengembangan frontend dan backend, serta teknologi kuantum. Situs ini dibuat yang berfokus pada penyediaan solusi efisien dan inovatif dalam transformasi digital.

## Fitur

- **Home**: Menampilkan portofolio dan informasi proyek secara keseluruhan.
- **About**: Menampilkan informasi detail tentang pembuat, termasuk media sosial dan statistik GitHub.
- **Contact**: Fitur bagi pengguna untuk mengirim pesan atau feedback.
- **Project**: Menampilkan project dan karya-karya digital.
- **Admin Panel**: Mengelola database melalui halaman CRUD (Create, Read, Update, Delete).
- **Profil Pengguna**: Pengguna dapat mendaftar, login, dan mengedit informasi pribadi.
